# EdgeOS UI 样式规范

完整 UI 设计规范以 **[EdgeX SaaS v3.0 样式参考](../operations/样式参考.html)** 为唯一权威版本（Design Tokens、布局、组件、北向页、Dashboard 等）。

EdgeOS 集成页面（edgeOS MQTT/NATS 北向通道、配置弹窗等）复用主规范中的通用样式：

| 场景 | 主规范章节 |
|------|-----------|
| 北向列表与配置弹窗 | §4.10 北向接口页 |
| 帮助弹窗 | `northbound-help-modal`（§4.10） |
| 表单与表格 | §4.3 Form · §4.2 Table |

EdgeOS 专有 UI 条目暂无独立扩展；新增样式应直接更新主规范 `docs/operations/样式参考.md`。
