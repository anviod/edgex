| # | 场景 | 必测 | 计划时长 | 执行日期 | 结果 | 备注 |

| S2 | **网络断开恢复** | ✔ | | | ☐ 通过　☐ 失败 | 拔线 / 防火墙 / 路由切换 |

| S5 | **超时** | ✔ | | | ☐ 通过　☐ 失败 | 响应超时与 CB 联动 |

**场景通过定义（每项）：** 采集恢复、无 channel 级 Offline（除非预期）、同通道其他设备不受影响（共享链路协议）、无 panic / 死锁。量化阈值与 PASS/WARN/BLOCK 裁量见 [G-Industrial — 发布判定标准](../RELEASE_GATE.html#g-industrial--工业验证门禁)。

## 6. G-Industrial 门禁判定

对照 [RELEASE_GATE — G-Industrial 发布判定标准](../RELEASE_GATE.html#g-industrial--工业验证门禁)：

### 6.1 检查项

| 检查项 | 标准 | 本报告结论 |
| --- | --- | --- |
| 协议覆盖 | 本协议联机/仿真验证完整；长跑时长符合协议分级（24h/72h） | ☐ 满足　☐ 不满足 |
| 异常场景 | **S2–S6 全部通过**（单项 FAIL → 本协议 BLOCK） | ☐ 满足　☐ 不满足 |
| 长时联机 | **S1** 达到协议要求时长且稳态指标达标 | ☐ 满足　☐ 不满足 |
| 量化指标 | §4 硬指标无 BLOCK 项（Lag P95、失败率、miss deadline、恢复时间等） | ☐ 满足　☐ 不满足 |
| 三件套 | 设计 + 代码 + **本报告** 已归档 | ☐ 满足　☐ 不满足 |

### 6.2 裁量级别

| 级别 | 触发条件（摘要） | 勾选 |
| --- | --- | --- |
| **PASS** | S1–S6 全过；§4 指标均为 PASS；无阻塞级偏差 | ☐ |
| **WARN** | 无 BLOCK；存在可解释偏差（如仅仿真、Lag WARN 档、单次恢复略超上限）且 §5 已跟踪 | ☐ |
| **BLOCK** | 任必测场景 FAIL；panic/死锁/跨设备污染；指标 BLOCK 档；CB 不可恢复 | ☐ |

**本协议 G-Industrial 结论：** ☐ **PASS**　☐ **WARN**（注明：`________`）　☐ **BLOCK**

> §6.2 为 **BLOCK** 时，§7 不得选择「准予纳入 Release 工业验证范围」。
