# B1 时间戳核验报告（Modbus TCP）

- 场景：同通道 3 台从站（slave-1/2 正常，slave-3 恢复后在线）
- 方法：采集层 PollStart 日志，统计同一轮三台设备的触发时间差 Δt
- 目标：任意两个设备采集时间间隔 Δt ≥ 10ms
