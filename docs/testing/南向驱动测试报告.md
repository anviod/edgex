## 五、已知缺口与风险

| 项目 | 说明 | 建议 |
| :--- | :--- | :--- |
| Profinet IO 真实设备 | 当前仅仿真模式与编解码测试，无真实 PNIO 从站 CI | 联机环境补充 RPC/IO 数据交换测试 |
| SNMP / ICE104 / DL/T645 联机 | 单元测试均为 Mock，无真实设备/模拟器 CI | 参考 [联机测试方案](../TODO/联机测试方案.html) |
| ShadowCore 基准 | integration 标签下编译冲突 | 合并 mock 定义或拆分 build tag |
| 覆盖率 | 主驱动包 27–66%（因协议而异） | 持续补充 scenario 与集成测试 |
| Profinet IO / KNX 硬件依赖 | 完整协议栈需特定网卡/组播环境 | 文档标注，生产部署前联机验证 |
