# 南向驱动测试报告

> **测试日期**: 2026-07-04  
> **环境**: macOS (darwin/amd64), Go toolchain, `CGO_ENABLED=0`

## 一、测试执行摘要

```bash
CGO_ENABLED=0 go test ./internal/driver/... -short -count=1 -cover
```

**未达 70% 说明**（自动化 `-short` 上限，非功能缺陷）：

| 驱动 | 主要未覆盖路径 | 建议 |
| :--- | :--- | :--- |

| Omron FINS | UDP backend、完整 FINS 帧 I/O | Mock PLC 已覆盖 TCP 主路径 |
| SNMP | v3 真实 UDP 探测、`GetBulk` 联机 | hook Mock 已覆盖调度读写 |
| ICE104 | `readLoop` / 入站 I 帧、`transport` 长连接 | integration tag + [test/ice104-goiecp5-server/](../../test/ice104-goiecp5-server/) |
| Profinet IO | `rpc.go` 真实 PNIO TCP | CI 使用 `simulation: true` |
| BACnet | `client.go` 网络收发、分段 APDU | Mock Client 已覆盖驱动 API |

## 五、本地运行指南

```bash
# 全量驱动（CI 推荐）
CGO_ENABLED=0 go test ./internal/driver/... -short -count=1 -cover

# 单驱动
CGO_ENABLED=0 go test ./internal/driver/snmp -short -v -count=1

# 集成 / 手工（非默认 CI）
CGO_ENABLED=0 go test -tags=integration ./internal/driver/ice104/... -count=1
CGO_ENABLED=0 go test -tags=manual ./internal/driver/bacnet/... -count=1
```

> **macOS 说明**：`internal/server`、`internal/sync` 等 TLS 包在旧 SDK 上可能链接失败；`internal/driver/...` 已验证 PASS。
