栅格化绝对对齐（Grid Alignment）：对画布的纵向（Y轴）和横向（X轴）空间建立严格的数字栅格，确保所有同级节点的中心点、连线的起点和终点都在同一像素刻度线上。

线段绝对不穿模（No Intersections / Overlaps）：优化节点布局与连线路径，确保数据流、信号线与任何矩形、文本或其他图形的边缘保持完美的像素级安全距离，绝对不发生重叠或穿模。

保持单条水平/垂直直线（Orthogonal Routing）：尽可能将连接线设计为直连的单条水平线或标准折线，消除斜线带来的凌乱感，保证视觉上的绝对工整。

间距完美对称（Symmetrical Spacing）：节点之间的上下、左右间距经过精确计算，实现像素级的完全等距与对称。

丰富与美化内容（Content Enrichment）：在保持极致工整的前提下，适当引入现代感的配色、渐变、阴影效果、微图表、状态指示灯或动态光点（AnimateMotion），提升整体的科技感与专业度。

纯净 SVG 格式输出：严格按照标准代码块格式输出可直接使用的标准 SVG 源码。
