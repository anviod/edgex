# 二、测试方法（Test Methods）

## 1️⃣ 单元测试（Unit Test）

**目标：** 验证报文构造、解析、长度校验逻辑。

**方法：**

* Table-driven tests
* Mock 网络接口
* 字节级断言

**覆盖：**

* BuildWhoIs()
* ParseIAm()
* Length validation

---

## 2️⃣ 集成测试（Integration Test）

**目标：** 验证完整发现流程与业务联动。

**方法：**

* 本地 UDP Socket + 模拟 BACnet 设备
* 或使用 BACnet 模拟器（YABE、BACnet Stack Demo）

**覆盖：**

* DiscoverDevices()
* RegisterDevice()
* AutoGeneratePoints()
* StartPolling()

---

## 3️⃣ 抓包对照验证（Protocol Compliance Test）

**目标：** 验证驱动发出的报文与抓包报文完全一致。

**方法：**

* Wireshark 抓包
* 对比 BVLC / NPDU / APDU 字段

---

## 4️⃣ 异常注入测试（Fault Injection）

**目标：** 验证系统在异常输入下的稳定性。

**方法：**

* 构造截断包、乱序包、重复包、畸形包
* 模拟网络丢包、延迟、风暴

---

# 三、验证标准（Acceptance Criteria）

| -------------- | ------------------ |

4. 所有 **性能测试项**：满足阈值要求
